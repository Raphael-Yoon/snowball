# 운영평가 모달 레이아웃 재배치

## 작업 일시
2025-12-02

## 변경 목적
설계평가 관련 항목을 위로 배치하고 운영평가 항목을 아래로 배치하여 사용자가 평가 흐름을 더 직관적으로 이해할 수 있도록 개선

## 변경 사항

### 이전 순서
1. 통제 정보
2. 당기 발생사실 없음
3. 모집단 업로드
4. 표본 크기 입력
5. 표본 라인 테이블
6. 전체 결론
7. 검토 의견 (운영평가)
8. 예외사항 세부내용
9. 개선계획
10. **설계평가 이미지** (하단)
11. **운영평가 이미지**
12. 수동통제 엑셀
13. 당기 발생사실 없음 사유

### 변경 후 순서
1. 통제 정보
2. **설계평가 이미지** ⬆️ **위로 이동**
3. 당기 발생사실 없음
4. 모집단 업로드
5. 표본 크기 입력
6. 표본 라인 테이블
7. 전체 결론
8. 검토 의견 (운영평가)
9. 예외사항 세부내용

11. **운영평가 이미지** (그대로 유지)
12. 수동통제 엑셀
13. 당기 발생사실 없음 사유

## 주요 개선 효과

### 1. 평가 흐름의 명확화
- **설계평가 정보** (통제 정보 + 설계평가 이미지) → **운영평가 입력** (표본, 의견 등) 순서로 자연스러운 흐름

### 2. 참고 자료의 접근성 향상
- 설계평가 이미지를 상단에 배치하여 운영평가 입력 시 쉽게 참고 가능
- 스크롤 없이 설계평가 결과를 확인하면서 운영평가 수행 가능

### 3. 논리적 그룹화
- **상단**: 설계평가 관련 정보 (읽기 전용)
- **중단**: 운영평가 입력 항목 (입력 필드)
- **하단**: 운영평가 증빙 및 부가 정보 (업로드 섹션)
